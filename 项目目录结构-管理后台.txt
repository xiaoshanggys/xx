管理后台目录结构示例：

/
├─ src/
│   ├─ views/            # 页面视图
│   ├─ components/       # 公共组件
│   ├─ router/           # 路由配置
│   ├─ store/            # 状态管理
│   ├─ api/              # API接口封装
│   ├─ utils/            # 工具函数
│   ├─ assets/           # 静态资源
│   └─ main.js           # 入口文件
├─ public/               # 静态资源
├─ package.json          # 依赖配置
└─ ...
